# BetterLogging

Better logging defaults.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "better_logging", github: "everydayhero/better_logging"
```

And then execute:

    $ bundle

## Usage

When `ENV["BETTER_LOGGING"]` is set logs will be sent to stdout and formatted as
JSON.
